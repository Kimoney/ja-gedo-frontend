# Ja Gedo Safaris Frontend

## Built with Vite

## How To Start

1. Clone the Repo
2. Install dependencies with
   ```bash
   npm install
   ```
3. Start the development server
   - On Linux/Mac:
     ```bash
     npm run dev
     ```
   - On Windows:
     ```bash
     npm run dev
     ```
4. Open your browser and navigate to `http://localhost:5173` (or the port specified in the terminal)
5. Enjoy!!
